## Open-Source Projects

### [libedhoc](https://github.com/kamil-kielbasa/libedhoc)
A C implementation of the EDHOC protocol (RFC 9528) — a lightweight authenticated key exchange designed for constrained devices. Provides mutual authentication, forward secrecy, and identity protection. Verified against RFC 9529 test vectors, with 635+ tests and 92.8% line coverage.

`c` · `iot` · `security` · `coap` · `cryptography` · `embedded` · `authentication` · `ietf` · `zephyr` · `key-exchange` · `oscore` · `edhoc` · `rfc9528` · `rfc9529` · `constrained-devices`

### [aeternusdb](https://github.com/kamil-kielbasa/aeternusdb)
An embeddable, persistent key-value storage engine built on a Log-Structured Merge Tree (LSM-tree) architecture. Written in pure Rust with write-ahead logging, automatic background compaction, Bloom filter lookups, MVCC range scans, CRC32 integrity, and crash recovery.

`rust` · `database` · `nosql` · `storage-engine` · `embedded-database` · `systems-programming` · `lsm-tree` · `key-value-store`

### [ubi](https://github.com/kamil-kielbasa/ubi)
Unsorted Block Images (UBI) support for Zephyr RTOS. Provides volume management for raw flash devices with wear-leveling, bad block handling, and dynamic volume resizing — bringing a battle-tested Linux concept to the embedded world.

`c` · `flash` · `embedded` · `virtualization` · `ubi` · `zephyr` · `zephyr-rtos`
